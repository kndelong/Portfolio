To create our specified subset of 10 users that performed the most actions in 2016 we start by taking the subset of the year 2016. Next, we group the subset by their user ID's; since each row in the dataframe consitutes an action we can count each line in the dataframe as an "action". This allows us to use the count() function as our aggragator (similar to SQL) this yields each user's cumulative actions for the year of 2016. In this action the dataframe is sorted in descending order and cut to the first 10 users.

Once we have the subset of the top 10 users we are able to subset the dataframe containing only the year 2016 by user and month. This yields our targeted data, however the data is transposed and we need to unstack the data to get our desired result. Finally, we count the actions in the final aggragated dataframe and check that the match our first cumulative count.

The relevant data is outputted to a .csv and the nessessary plot is generated using the Pandas/Matplotlib interdependency. To yield the correct figure we need to again take the transpose of the dataframe so the data is in it's proper time-series orientation.

# Subset the dataframe to the year 2016
df_2016  = df[df.date.dt.strftime('%Y') == '2016']

# Find the top 10 users by grouping on their ID's and counting their actions as the aggragate function
top_10 = df_2016.groupby(by = ['userID']).actions.count().sort_values(ascending=False).iloc[:10].index.to_list()

# Subset the dataframe by the top 10 users, then group by user and month again counting actions as the aggrate function
agg = df_2016[df_2016['userID'].isin(top_10)].groupby(['userID', df_2016.date.dt.strftime('%Y-%m')]).count()['actions'].unstack()

# Add the total actions taken as a new column
agg['Total'] = agg.sum(axis = 1)

# Sort the values in descending order
agg.sort_values(by = 'Total', ascending = False, inplace = True)

# verify that the Total matches the count found in top_10
df_2016.groupby(by = ['userID']).actions.count().sort_values(ascending=False).iloc[:10].values == agg.Total.values

For the statistics about the data I provided two methods: the first using functions implemented by Scikit-learn and secondly using linear algebra.

This two-approach method allowed me to verify the results of the Sklearn methods. Since the data was a comparison of two entries in the dataset A and B are slices into the dataframe retrieving the top 1 and 2 users respectively. For the Sklearn methods, the relevant values are reported in the the diagonal of an array, this is a product of passing the Sklearn method a dataframe/array which is easier than passing each row individually as this ignores the requirement to reshape the features.
